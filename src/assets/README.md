# Image Guidelines

Keep images as light as possible while preserving enough detail for their role in the UI.

| Image type | Recommended size | Use case |
| --- | --- | --- |
| Small icons/thumbnails | Under 30 KB | UI elements, small avatars |
| Standard images | 70 KB - 200 KB | Blog photos, product images |
| Hero/full-width | 300 KB - 500 KB | High-impact banners, headers |
| Backgrounds | Under 600 KB | Full-screen textures or patterns |

Use the closest matching category for files in this folder:

| Folder | Typical image type |
| --- | --- |
| `people/` | Small avatars or standard images |
| `reviews/` | Product images or thumbnails |
| `site/` | Hero, full-width, or background images |
